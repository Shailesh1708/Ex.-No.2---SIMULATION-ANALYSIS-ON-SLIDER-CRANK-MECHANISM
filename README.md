# Ex. No. 2- SIMULATION ANALYSIS ON SLIDER CRANK MECHANISM

## DATE: 04.03.2026

## AIM:
###   To determine & simulate the displacement & velocity analysis for the given slider crank mechanism. 
###   An engine mechanism is shown. The Crank CB = 200mm and the connecting rod BA = 600mm. In the position shown, the crankshaft has a speed of 50 rad/s and an angular acceleration of 800 rad/s2


## REQUIREMENTS:
###   ●	Virtual Labs.
###   ●	Processor: Minimum 1.5 GHz
###   ●	RAM: Minimum 512 MB
###   ●	Operating System: Windows XP, Windows Vista, Windows 7, Windows 8 or higher.
###   ●	Dependencies: Microsoft .Net 2.0 framework
###   ●	Mini Drafter.
###   ●	Geometry instruments.

## Procedure

### 1. Open the Virtual Lab for the Slider Crank Mechanism (Acceleration Analysis) experiment.

### 2. Identify the given parameters from the problem statement.

###    Length of crank CB (r) = 200 mm

###    Length of connecting rod BA (l) = 600 mm

###    Angular velocity of crank ω = 50 rad/s

###    Angular acceleration of crank α = 800 rad/s²

### 3. Determine the crank angle (θ) from the position shown in the diagram.
### 4. This angle is measured between the crank CB and the line of stroke of the slider.

### 5. Enter the input parameters in the simulation fields of the Virtual Lab:

###    Crank length r = 200 mm

###    Connecting rod length l = 600 mm

###    Crank angle θ = 120

###    Angular velocity ω = 50 rad/s

###    Angular acceleration α = 800 rad/s²

### 6. After entering the values, run the simulation to perform the displacement, velocity, and acceleration analysis of the slider crank mechanism.

### 7. The simulation will automatically calculate the velocity and acceleration of the slider and connecting rod based on the given crank motion.

### 8. Observe and record the results displayed by the simulation.

### 9. Analyze the graphical output such as velocity and acceleration diagrams provided by the virtual lab.

### 10. Note down the obtained results and include them in the GitHub report along with the simulation screenshot if required.
